What is HTTP?

The term HTTP stands for Hyper Text Transfer Protocol. The User tries to access the internet and sends a request to the web server. The Web server retrieves the request and sends an response and the user recieves it and sees the output of the browser

What is Cache Poisoning?

Cache Poisoning is also known as DNS spoofing. Cache poisoning is when corrupted false information is entered into an DNS cache. As an result the DNS will return an incorrect resposne and push users to the wrong website.
